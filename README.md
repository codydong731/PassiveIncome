 # Passive Income Dashboard
 
 Simple, self-contained dashboard for projecting passive income based on
 investment categories and return assumptions. Runs in any modern browser
 with no build step.
 
 ## Files
 - `index.html`
 - `styles.css`
 - `app.js`
 
 ## Run locally
 Open `index.html` in your browser.
